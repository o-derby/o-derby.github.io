Rythym Game

COMING SOON: I’m planning to build a piano-style rhythm game where players hit keys in time with the beat (because I love the rythym game Project SEKAI by SEGA). The project will focus on smooth gameplay mechanics, timing accuracy, and fun visual feedback for correct notes. I’m thinking of starting with a simple web-based version using JavaScript, then expanding to include a library of songs and difficulty levels. This will be both a coding challenge and a chance to create something interactive and creative.
